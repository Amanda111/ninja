# Ninja

Ninja powered front-end development server. 

## Features
+ **Generate routes automatically**
+ **Interpret Nodejs template**
+ **Jinja syntax supported(swig)**
+ **Livereload for HTML&CSS**
+ **Proxy mode**
+ **Webpack compatible**


## Install
```
npm install -g ninja_cli
```
